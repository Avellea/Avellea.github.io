WHAT IS MAYUKO?

Mayuko is a Discord bot for all things anime. I’ve tried to give her a bit of personality by making the responses feel a bit more “human” like, however looking back on making her, it feels a bit strange. For example, the $info command returns the following.

Hi there! I’m Mayuko, but you can call me Ma-chan if you’d like. I’m a huge otaku with nearly infinite knowledge of all things anime, including the magic numbers~! Since I’m not actually human, you can look at my code on GitHub. I’ve gotta get back to what I was watching now, but if you need me, just type $help and I’ll be right over!

WHAT DOES MAYUKO DO?

Not a whole lot in her current state. Right now, she can really only access public Anilist information, such as anime name, airing status, episodes, and images. She can also access nekos.life for some fun anime images. SFW and NSFW, though you didn’t hear that from me ;)

Lastly, she can access basic information from nhentai.net. Tags, a thumbnail, and popularity can be retrieved.

WHY DID YOU MAKE SUCH AN ABOMINATION?

Well it really started out as a joke… Some friends jokingly said they wanted a Discord bot to easily find hentai, so I thought to myself, “Alright, see you in a week.” A little over a week later, Mayuko was in her very early state with only basic commands to access the service that I keep crossing out. Over time, I found myself wanting to work more with the Discord API, so I kept adding small features. I think it started out with the $waifucommand. It pulls a random character from MyWaifuList.moe, builds a basic embed, and sends it back to the user. Next came the $anisearch and $charsearch commands. Those just hit the Anilist API for some info.

I WANT THIS ON MY DISCORD SERVER! HOW CAN I GET IT?

I will not be publicly hosting it. It made it for personal use, and it will stay that way. However, you can self-host it if youd like. The code is on my Github profile. Follow the instructions in the README and you’ll be good to go.
